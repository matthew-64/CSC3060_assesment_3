# Artificial Intelligence and Data analysis
This assingment uses the R language to analise and catorgise black and white, hand-drawn images of living and non-living things. The image types are as following:  

Living images:
* cherry
* flower
* banana
* pear

Non-living images:
* envolope
* wineglass
* pencil
* golfclub

For all sections, Vaurios histograms and graphs will be generated to illustrate the findings. Key exact features will be printed to the console.

## Section 1
This section used logistic regression [logistic regression](https://en.wikipedia.org/wiki/Logistic_regression) using precalculated features to calculate the probability of each image being living or non-living

To run section 1 execute the following command:  
`Rscript section_1.r`

## Section 2
This section used [k-nearest-neighbour classification](https://towardsdatascience.com/machine-learning-basics-with-the-k-nearest-neighbors-algorithm-6a6e71d01761) for precalculated features to classify all of the 8 images.

To run section 2 execute the following command:  
`Rscript section_2.r`

## Section 3
This section used logistic regression [Desision Tree](https://dataaspirant.com/2017/01/30/how-decision-tree-algorithm-works/) algrothim and [Random Forest](https://towardsdatascience.com/understanding-random-forest-58381e0602d2) using precalculated features to classify all of the 8 images.

To run section 3 execute the following command:  
`Rscript section_3.r`
